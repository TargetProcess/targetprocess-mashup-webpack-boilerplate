
# Targetprocess Mashup Webpack Boilerplate

[Webpack](http://webpack.github.io) boilerplate to create Targetprocess mashup.

## Usage

Clone repository. 

Run `npm install`.

Edit `webpack/*.js` to make suitable for your needs. Edit source code in `src` folder.
